简单的名片管理系统
=========================

Simple Card Manager System for J2EE Homework.  

### 简介
初衷是J2EE的期中作业，现在作业的要求还没有出来，只能根据自己的想象的需求来做。  

### 主要技术
主要使用J2EE开发，应用了Struts2框架，结合Bootstrap来优化界面。恩，现在就用了这么多。  

### 进度

基本大功能已经实现，接下来要进行用户体验的优化了。  

####当前已实现的功能有：  

1. 登录
2. 注册
3. 添加名片
4. 修改名片
5. 删除名片到回收站
6. 从回收站彻底删除名片

####还想实现的功能有：

1. 注册用户名验证
2. 等等
